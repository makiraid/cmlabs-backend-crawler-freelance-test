Catatan Penting

  • Dalam menggunakan beberapa tool yang mengharuskan Anda memasukkan URL, pastikan itu valid. Jika tidak, tool tidak akan berfungsi dengan baik.
  • Jika Anda menjalankan pengujian pada satu tool dan ini menunjukkan hasil yang berbeda pada waktu yang berbeda, perlu diingat bahwa itu mungkin karena pembaruan Google.
  • Jika Anda kehilangan koneksi internet di tengah pengujian tool, kami sarankan Anda menyegarkan halaman dan mencoba menjalankan pengujian lagi.
  • Tool ini disediakan secara gratis dan tidak memiliki batasan untuk pengguna mana pun.
  • Jika Anda menemukan bug atau masalah dalam menggunakan tool ini, silakan hubungi tim kami di obrolan langsung di sudut bawah cmlabs.co.

Cara Penggunaan

WORD COUNTER

  1. Salin artikel yang ingin Anda periksa.
  2. Tempel di bidang yang tersedia.
  3. Anda akan mendapatkan hasil untuk jumlah karakter, kata, kalimat, paragraf dan waktu membaca.
  4. Di sebelah kanan, per 1 kata, 2 kata, hingga 5 kata, Anda juga akan mendapatkan kepadatan keyword.

JUDUL & META DESKRIPSI CHECKER

  1. Salin URL halaman web yang ingin Anda analisis.
  2. Tempel di bidang URL yang tersedia.
  3. Klik tombol Dapatkan Judul & Meta Desc.
  4. Judul yang dihasilkan dan deskripsi meta halaman akan tersedia di bidang yang tersedia. Anda dapat mengedit langsung di bidang itu.

SKEMA GENERATOR JSON-LD

Breadcrumb

  1. Isi nama Halaman dan URL.
  2. Jika Anda ingin menambahkan lebih banyak, klik tombol "Tambahkan URL".
  3. Setelah selesai, periksa kode di kolom kanan untuk memastikan apakah memenuhi syarat di Rich Result Google.
  4. Terakhir, Anda dapat menyalin kode tersebut ke situs web Anda.

FAQ

  1. Isi pertanyaan dan jawaban di kolom yang tersedia.
  2. Jika Anda ingin menambahkan lebih banyak, klik tombol Tambah FAQ Lain.
  3. Setelah selesai, periksa kode di kolom kanan untuk memastikan apakah memenuhi syarat di Rich Result Google.
  4. Terakhir, Anda dapat menyalin kode tersebut ke situs web Anda.

How-to

  1. Isi kolom yang diperlukan untuk menjelaskan langkah Anda.
  2. Jika Anda ingin menambahkan lebih banyak, klik tombol +.
  3. Setelah selesai, periksa kode di kolom kanan untuk memastikan apakah memenuhi syarat di Rich Result Google.
  4. Terakhir, Anda dapat menyalin kode tersebut ke situs web Anda.

Job Posting

  1. Isi kolom yang diperlukan untuk menjelaskan deskripsi pekerjaan Anda.
  2. Jika Anda ingin menambahkan lebih banyak, klik tombol +.
  3. Setelah selesai, periksa kode di kolom kanan untuk memastikan apakah memenuhi syarat di Rich Result Google.
  4. Terakhir, Anda dapat menyalin kode tersebut ke situs web Anda.

Person

  1. Isi kolom yang diperlukan untuk menjelaskan identitas orang tersebut.
  2. Jika Anda ingin menambahkan lebih banyak, klik tombol +.
  3. Setelah selesai, periksa kode di kolom kanan untuk memastikan apakah memenuhi syarat di Rich Result Google.
  4. Terakhir, Anda dapat menyalin kode tersebut ke situs web Anda.

Produk

  1. Isi kolom yang diperlukan untuk menjelaskan produk yang dijual.
  2. Jika Anda ingin menambahkan lebih banyak, klik tombol +.
  3. Setelah selesai, periksa kode di kolom kanan untuk memastikan apakah memenuhi syarat di Rich Result Google.
  4. Terakhir, Anda dapat menyalin kode tersebut ke situs web Anda.

Recipe

  1. Isi kolom wajib untuk menjelaskan resep, bahan dan ulasan.
  2. Jika Anda ingin menambahkan lebih banyak, klik tombol +.
  3. Setelah selesai, periksa kode di kolom kanan untuk memastikan apakah memenuhi syarat di Rich Result Google.
  4. Terakhir, Anda dapat menyalin kode tersebut ke situs web Anda.

PAGE SPEED

  1. Masukkan link yang ingin Anda periksa di kolom. Pastikan link tersebut ditulis dengan format seperti ini https://domain.com/.
  2. Kemudian, untuk memulai proses analisis, tekan tombol "Analisis URL".
  3. Tunggu hingga perayapan selesai dan popup akan menutup dengan sendirinya.
  4. Hasilnya akan ditampilkan pada diagram lingkaran yang tercantum di bagian bawah.

GENERATOR SITEMAP

  1. Isi URL di kolom yang tersedia.
  2. Klik tombol Buat XML.
  3. Tunggu proses perayapan.
  4. Unduh peta situs XML.

MOBILE FRIENDLY TEST

  1. Ketik link yang ingin Anda analisis di kolom yang tersedia. Ketik link dengan format berikut http://domain.com/.
  2. Kemudian untuk memulai proses, tekan tombol "Periksa URL".
  3. Tunggu hingga perayapan selesai dan popup akan menutup dengan sendirinya.
  4. Saat perayapan selesai, hasilnya akan muncul di bagian bawah.

SSL CHECKER

  1. Isi URL di kolom yang tersedia.
  2. Klik tombol Periksa SSL.
  3. Hasilnya akan diberikan pada kolom di bawah ini.

ROBOTS.TXT GENERATOR

  1. Setel default izinkan untuk mengizinkan perayap mengindeks situs web.
  2. Tentukan penundaan perayapan jika Anda ingin menunda waktu pengindeksan.
  3. File robots.txt harus berada di root host situs tempat file akan diberlakukan. Misalnya, untuk mengontrol perayapan semua URL di http://www.example.com/, file robots.txt harus ditempatkan di http://www.example.com/robots.txt. File tidak boleh berada dalam subdirektori (misalnya, di http://example.com/pages/robots.txt), jika tidak biarkan kosong.
  4. Pilih akses untuk memilih izinkan atau larang.
  5. Pilih agen pengguna yang ingin Anda targetkan.
  6. Pilih direktori yang ingin Anda tuju.
  7. Lalu buat robot.

REDIRECT CHECKER

  1. Isi URL di kolom yang tersedia.
  2. Pilih user agent yang ingin Anda gunakan.
  3. Klik tombol Periksa URL.
  4. Hasilnya akan diberikan pada kolom di bawah ini.

TECHNOLOGY LOOKUP

  1. Isi URL di kolom yang tersedia.
  2. Klik tombol Cari.
  3. Hasilnya akan diberikan pada kolom di bawah ini.
  4. Catatan: Teknologi yang terdaftar dibagi per fungsionalitas untuk menampilkan klasifikasi.

LINK ANALYZER

  1. Isi URL di kolom yang tersedia.
  2. Klik tombol Analisis.
  3. Hasilnya akan diberikan pada kolom di bawah ini.
  4. Catatan: Tautan internal adalah tautan yang merujuk ke halaman dalam domain yang sama. Tautan eksternal adalah tautan yang merujuk ke halaman di domain berbeda.

PEMERIKSA HREFLANG

  1. Isi URL di kolom yang tersedia.
  2. Klik tombol Periksa URL.
  3. Hasilnya akan diberikan pada kolom di bawah ini.
  4. Catatan: Bahasa dan Wilayah mengacu pada bahasa default situs web Anda. Jika "tidak ditentukan", hubungi developer Anda untuk menyiapkan bahasa href.
