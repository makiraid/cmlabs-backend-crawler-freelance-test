JSON-LD Person Schema Generator Tool

Writen by cmlabs

|

Published at Oct 13, 2023

What is the JSON-LD Person Schema Generator Tool?

The person schema generator is a tool that you can use to create schema markup that contains personal information about an individual. This type of schema helps to generate structured data for each person or individual.

Maybe, you have seen search results for celebrities, political figures, athletes, and more displayed in a special box in the SERP. Well, that's what the Person markup schema looks like. Not only applies to famous people, but you can also use this scheme for personal branding purposes.

In using this tool, you will be asked to enter general personal information such as the person's full name, URL, URL of a photo or image about the person, social media profiles owned, job data, to the company name.

Functions of Person Schema Generator

This tool has the main function of generating schema markup code of the Person type. With this tool, users do not need to create it manually, our system will generate code directly when you fill in the required information and data.

This information will help search engines distinguish between different people with the same name. Not only that, this scheme will help you to appear in the SERPs and improve SEO implementation.

In addition, generating schema markup with this tool will bring a number of benefits, namely:

  • provide person-specific information in the search result
  • variety of personal information that can be displayed
  • highlight the search results

How Schema Markup Affects SEO

Schema markup is important because it makes your website look better in search results. Thanks to this schema, additional elements such as photos, or brief information are displayed as rich snippets in the SERPs.

Users are more likely to click on those results because they are more interesting and contain the information they need. So, Schema markup can help keep up with your competitors.

Example of Generated Person Schema Markup

Here we show an example of a script that was successfully obtained through this free tool.

                
                    <script type="application/ld+json">
                        {
                            "@context": "https://schema.org/",
                            "@type": "Person",
                            "name": "Jane",
                            "url": "https://www.linkedin.com/",
                            "image": "https://images.unsplash.com",
                            "sameAs": [
                                "jane_isme"
                            ],
                            "jobTitle": "SEO specialist",
                            "worksFor": {
                                "@type": "Organization",
                                "name": "SEO organizataion"
                            }
                        }                            
                    </script>

That's a brief explanation of what the JSON-LD Person Schema Generator from cmlabs is. We have explained the definition, function, and how the markup schema can play a role in your SEO implementation. Try now!

How to Use Person Schema Generator?

The steps to generate a person schema using this tool is as follows:

Go to JSON-LD Person Schema Generator Page

Please open the JSON-LD Person Schema Generator page to start your steps in automatically creating a person schema markup. Make sure you have selected "Person" in the type of schema markup you want to create.

Figure 1: Display of the person schema generator page from cmlabs.

Input Your Name

First of all, input your name in the space provided. Make sure you have entered the correct name format.

Figure 2: Enter a name in the column provided to create a person schema markup.

Add your URL

Next, you can add any URL, which can be a personal URL or a URL from a company that you own. In adding URL data make sure you have input the right URL and not a broken link.

Figure 3: Enter the URL in the space provided to create a person schema markup.

Enter Image URL

With the image URL your personal profile will be displayed with certain images in search results. Make sure to enter a URL that matches your image file path.

Figure 4: Enter the image URL in the space provided to create a person schema markup.

Add Social Media Profile

Next, add a social media profile. There are many social media options that you can choose from in this tool. After determining the choice of social media, you need to enter the URL of your social media in the column provided.

Figure 5: Social profile options that can be added to create person schema markup.

Complete Job Data

You'll also need to include job data. Write the name of your current job in the column provided. Then, make sure there are no typos in your data.

Figure 6: Enter the type of work in the column provided to create a person schema markup.

Put Company Name

Next, add the name of the company where you currently work. As with the job title, make sure to enter the company name in the correct writing format.

Figure 7: Enter the company name in the column provided to create a person schema markup.

Copy Generated Person Schema Markup

If you have completed the required attributes, then you will see a preview of the schema markup on the right side of the page. You can copy the schema markup directly with the copy button and paste it into your program code.

Figure 8: Person schema markup copy options

Check JSON-LD Person Schema Script Validation

With the Person schema generator, you can also check the validity of the JSON-LD script you are using. How to check it is quite easy, just press the check option contained in the preview of the resulting script.

When you press the button, you will be redirected to Google's Rich Result Test page. On that page, you can check if the generated schema markup is working properly in Google search.

Figure 9: Person schema markup validation check options

Reset Schema Generator

This tool has options that make it easier for you to create a new Person schema. Just click the "Reset" option and delete all the data that was added previously. Next, you can create a new script.

Figure 10: Data reset option added to Person schema generator.

That's a brief discussion on how to create a person schema using the JSON-LD person schema generator tool from cmlabs. These tools can help you generate schema markup quickly and efficiently. Try this tool now!
